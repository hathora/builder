# Hathora Builder - Reference: CLI

```
% hathora --help
hathora <command>

Commands:
  hathora build                         Builds the project  [aliases: b]
  hathora cloud                         Interact with Hathora Cloud
                                                            [aliases: c]
  hathora create-client <template>      Creates a client subdirectory
  <name>
  hathora create-plugin <lib> <type>    Creates a plugin
  hathora dev                           Starts the server in development
                                        mode   [aliases: development, d]
  hathora generate                      Regenerates the types from
                                        hathora.yml    [aliases: gen, g]
  hathora init                          Creates a new Hathora project
                                       [aliases: initialize, initialise]
  hathora install                       Install hathora dependencies
                                                            [aliases: i]
  hathora save <stateId> <saveName>     Creates a named save game from a
                                        specific state id
                                                 [aliases: gamesave, sv]
  hathora start                         Starts the Hathora server
                                                        [aliases: up, s]
  hathora completion                    generate completion script

Options:
  --help     Show help                                         [boolean]
  --version  Show version number                               [boolean]
```
